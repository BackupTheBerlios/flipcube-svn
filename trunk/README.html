FlipCube

Description

A 3D screensaver built on Java3D that animates a folding and rotating "flip cube" whose faces are adorned with the pictures of your choice.

Dedication

To the Creator of 3D space, of topology, and of light and vision.
Thank you for making us creators, in your image.

How to build

Prerequisites

  • Java SDK 1.4
  • Java 3D 1.3
  • Downloading the class docs for Java 3D is also recommended if you plan to do any development.
  • There is also a Java3D tutorial, but it is out of date in some sections, so it is good to have the class docs too.

Compiling

An elegant build process (e.g. with Ant) is one of our todo's. Fortunately, FlipCube is a simple program and not hard to compile by hand. You should be able to just do

   javac *.java

assuming that your Java SDK is set up right.

Building for a screensaver version and/or a Java Web Start version are on our to-do list.

How to run

Make sure Java 3D is installed right, then run

   java FlipCube

When we figure out how the screensaver version will work, we'll give instructions on how to run it too. (We do have some ideas though.)
